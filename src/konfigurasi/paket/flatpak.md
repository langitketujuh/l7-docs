# Flatpak sanboxing

## Konfigurasi flathub

Diharuskan menjalankan perintah ini saat pertama kali menggunakan flatpak atar tidak galat saat melakukan pemasangan aplikasi.

```sh
flathub
```

## Memasang perangkat lunak
